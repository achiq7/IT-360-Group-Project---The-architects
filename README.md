# Overview
We created a Forensics Timeline Builder tool that can be used to help accelerate investigations. Our tool will take a text/csv file, organize the contents chronologically, and give a written report of the logs. This is a quick alternative to manually organizing data. 

# Features
- Collection of large data files
- Parse through given log files
- Extracts the timestamps and data
- Sorts data chronologically for timeline analysis
- Exports organized data into a CSV file to furthere analyze
- Generate a PDF report of timeline
- Formattig of the report

# Setup Instructions
- Download timeline_builder.py file, create_report.py file, and ssh_logs.txt file all into the same folder
- Run timeline_builder.py 
- Run create_report.py
- View Forensics Timeline Builder pdf found in current director
